## WHAT IS BAT?

* BAT is a suite of tools for processing audio in creative ways.
* BAT is modular: each module performs a different kind of processing, and modules can be connected in chains to perform complex transformations
* BATcave provides central services, such as opening modules, saving workspaces and recording outputs
* BATcave is not necessary to use BAT - all BAT modules will also work on their own, or in a peer-to-peer basis with others.

## REQUIREMENTS

**BAT requires Max 8 to run.** If you are running Max 8 in demo mode, you will be able to use the software, but not modify the patches yourself. BAT will not run completely using Max versions earlier than 8 - but a few things might work. Mac 9 should be ok, but has not been tested.

BAT is only regularly tested on Mac ARM (Apple Silicon). Some things may not work on Mac Intel or Windows, but most things should be ok.

## INSTALLATION

**First, make sure you completely delete ALL old versions of BAT.**  
Next, drag the entire BAT folder to your Max Packages folder. The Packages folder will be one of the following...

(Max 8 Mac) ~/Documents/Max 8/Packages/ OR /Users/Shared/Max 8/Packages
(Max 8 Windows) My Documents\Max 8/Packages/ OR C:\ProgramData/Max 8/Packages\

## GETTING STARTED

After installation, BAT will be accessible from the Max 'Extras' menu. Open Max, then go to 'Extras' and choose 'BAT'. 
  
On launch, BATcave will open, and other modules can be chosen from there.  

To process a sound file, drag it anywhere into a BAT module window.  

To use input from another BAT module, select 'BATbuss' as the input.
